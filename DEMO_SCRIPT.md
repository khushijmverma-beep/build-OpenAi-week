# Three-minute demo

1. Open TextEdit. Explain that the keyboard invokes work while voice supplies the input.
2. Press Control-Option-D, dictate one literal sentence, press it again, and show local insertion.
3. Press Control-Option-K, speak a rough sentence with a correction, and show Smart Writing returning only polished prose.
4. Select a paragraph, ask for a concise professional rewrite, and show the Accessibility/clipboard-safe delivery result.
5. Press Control-Option-Q. Ask Jarvis to open Safari and another app. Point out streamed overlay state and tool receipts.
6. Ask Jarvis to remember a browser preference, create “start work setup,” then run it.
7. Request restart. Show the time-bound confirmation state and cancel it.
8. Close with native Swift/AppKit, local dictation, OpenAI Realtime, cost-aware Responses routing, Keychain, typed tools, and SQLite.

Do not demonstrate music playback or unverified system actions.
